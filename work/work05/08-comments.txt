program ini digunakan untuk melihat memory yang digunakan. cara menampilkannya dengan menggunakan malloc() dan system().
malloc() sendiri merupakan fungsi bawaan dari C yang digunakan untuk mengalokasikan memory yang dibutuhkan dan mengembalikannya dalam bentuk pointer.
sedangkan system() merupakan fungsi bawaan dari C yang digunakan untuk melewati sebuah perintah atau program yang ditentukan oleh command ke host yang digunakan untuk dieksekusi command processor dan kembali setelah command selesai dikerjakan
