# Github Account: filzav

1.	PID(Process Identifier) merupakan nomor unik sementara yang di-assign oleh kernel, sedangkan PPID(Parent Process Identifier) merupakan nomor unik sementara untuk mengidentifikasi parent process saat ini.
	getpid() digunakan untuk mendapatkan process ID saat ini, getppid() digunakan untuk mendapatkan parent process ID, sleep digunakan untuk mematikan proses yang sedang berjalan, wait digunakan untuk menunggu proses berjalan/menunggu proses dimatikan
	return values dari fork() adalah 0 ke child process dan mengembalikan process ID dari child process menuju parent process

2.	procStatus melakukan pengembalian status proses dari proses yang sedang dilakukan
	wait(null) digunakan untuk menunggu sampai state berubah pada child process
	fflush(null) digunakan untuk menghapus semua open output streams
