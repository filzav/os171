Global Variable Address merupakan variabel yang tidak memiliki batasan scope sehingga variabel tersebut bisa diakses oleh siapapun bahkan dengan program lain yang memanggil variabel tersebut sedangkan Local Variable Address merupakan variabel yang memiliki batasan scope sehingga variabel tersebut hanya bisa diakses oleh tempat dimana variabel tersebut dituliskan, jika dipanggil variabel tersebut diluar tempat ia dituliskan maka variabel tersebut tidak akan ditemukan.

2 a. Parameter flags pada fungsi open di kode tersebut adalah untuk membuat sebuah folder/file seperti yang diinginkan
b. Parameter mode pada fungsi open di kode tersebut adalah untuk mengeset permission bagi siapapun yang ingin mengakses folder/file tersebut
c. I/O merupakan kependekan dari Input dan Output dimana input merupakan memasukan sebuah data ke dalam program, sedangkan output merupakan menampilkan data pada layar, printer, atau file lainnya. Ada standard files pada c seperti standard input (stdin), standard output (stdout), dan standard error (stderr). Selain itu terdapat beberapa  macam fungsi untuk I/O seperti getchar(), putchar(), gets(), puts(), scanf() dan printf().
